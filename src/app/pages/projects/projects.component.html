Magic Formula Investing scraping

This project contains a web scraper that digs through the Magic Formula Investing website to gather a list of 50 undervalued stocks. With this list, I ping an financial API that helps me gather data on these stocks and writes all of the data to a MongoDB database. It is an amazing tool that helps me find new investments ideas for my stock portfolio. This is not investment advice!
